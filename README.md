# Análise Comparativa de Algoritmos de Ordenação

## 1. Descrição do Projeto

Implementação e comparação de 6 algoritmos de ordenação em Java:
- **Comb Sort**
- **Gnome Sort**
- **Bucket Sort**
- **Bubble Sort** (com flag de parada)
- **Selection Sort**
- **Cocktail Sort**
